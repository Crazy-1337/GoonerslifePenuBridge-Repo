# Goonerslife Penu Bridge (Plugin Repository)

A personal [Dalamud](https://github.com/goatcorp/Dalamud) plugin that gives you a fast, dark-themed overlay for toggling **GoonersLife+ v2.1 [Gooners.inc]** Penumbra mod options without ever opening Penumbra.

Author: Skylar Kaisa
Requires: (https://github.com/xivdev/Penumbra)

---

## 🚀 In-Game Installation

Dalamud supports custom third-party repositories. Follow these steps to install the plugin.

### Step 1: Open Dalamud Settings
Type the following chat command in-game to open the **Dalamud Settings** window:
```chat
/xlsettings
```

### Step 2: Add the Repository URL
1. Click the **Experimental** tab at the top of the window.
2. Scroll down to the **Custom Plugin Repositories** section.
3. Click the **`+`** button to add a new entry.
4. Paste the following URL into the text box:
   ```text
   https://Crazy-1337.github.io/GoonerslifePenuBridge-Repo/pluginmaster.json
   ```
5. Ensure the toggle next to the URL is **enabled (green)**, then click **Save**.

### Step 3: Install the Plugin
1. Open the plugin installer: `/xlplugins`
2. Switch to the **All Plugins** tab or use the search bar to find `Goonerslife Penu Bridge`.
3. Click **Install**.

### Step 4: First-Time Setup
1. Type `/qmp` to open the panel.
2. On first launch, select your active **Penumbra collection** (usually *Current* or your character's name).

## 🔄 Updating

Dalamud checks for plugin updates automatically. When a new version is released, you will see an **Update** button inside `/xlplugins` under the **Installed Plugins** tab.

## 🛠️ Troubleshooting

| Problem | Fix |
| :--- | :--- |
| **Plugin doesn't appear in the installer** | Verify the repo URL ends in `/pluginmaster.json` and the toggle is green. |
| **"Penumbra not found" error** | Install and enable Penumbra first, then reopen the panel. |
| **Mod options not showing** | Ensure GoonersLife+ is installed in Penumbra and the correct collection is selected. |
| **Validation warning in plugin list** | Ensure you are running the latest version of Dalamud. |
